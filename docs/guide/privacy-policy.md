为保障本服务的正常运行、安全维护、性能分析及违规审查，本平台可能收集和处理以下信息：

### 1.1 你主动提供的信息
当你通过邮件或其他方式联系本平台时，你可能主动提供以下信息：
- 邮箱地址；
- 联系内容；
- 问题描述、反馈内容；
- 你主动提供的其他信息。

### 1.2 在你使用服务过程中自动收集的信息
当你访问或调用本服务时，本平台可能自动记录以下技术信息或日志信息：
- 来源域名；
- 访问 IP 地址；
- 调用时间；
- 请求路径；
- 请求方法；
- 请求频率；
- 调用次数；
- 流量消耗；
- User-Agent、Referer、基础设备或浏览器特征；
- 响应状态码；
- 错误日志、安全日志、风控日志；
- 为保障系统稳定所必需的其他技术信息。

### 1.3 特别说明
根据本平台当前服务规则，本平台会记录你的**域名、使用流量、调用次数等信息，并存储六个月**。如果你对此介意，请停止使用本服务。

### 5.1 共享
